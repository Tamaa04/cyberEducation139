Cyber Security Tips

Tips Keamanan Digital

Pelajari berbagai tips sederhana namun sangat penting untuk melindungi akun, perangkat, dan data pribadi ketika menggunakan internet.

Tips Unggulan

Gunakan Password Yang Berbeda Untuk Setiap Akun

Menggunakan password yang sama untuk semua akun dapat meningkatkan risiko apabila salah satu akun berhasil diretas.

Pelajari Selengkapnya
Password

Gunakan Password yang Kuat

Kombinasikan huruf besar, huruf kecil, angka, dan simbol agar password sulit ditebak.

Phishing

Periksa Alamat Website

Pastikan URL menggunakan HTTPS dan berasal dari domain resmi sebelum login.

Privacy

Batasi Informasi Pribadi

Hindari membagikan data sensitif di media sosial maupun forum publik.

Network

Hindari WiFi Publik

Gunakan VPN jika harus terhubung ke jaringan WiFi umum.

Device

Selalu Update Sistem

Update sistem operasi dan aplikasi untuk menutup celah keamanan.

Device

Backup Data Secara Berkala

Simpan salinan data penting di cloud maupun harddisk eksternal.

Privacy

Atur Privasi Media Sosial

Batasi siapa saja yang dapat melihat informasi pribadi dan unggahanmu.

Password

Jangan Berikan Kode OTP

Kode OTP hanya untuk pemilik akun. Jangan pernah membagikannya kepada siapa pun.

Device

Gunakan Antivirus

Antivirus membantu mendeteksi dan mencegah serangan malware.

Checklist Keamanan

Sudahkah Kamu Melakukan Ini?

Checklist sederhana yang dapat membantu meningkatkan keamanan akun dan perangkatmu.

Mengaktifkan Two Factor Authentication
Rutin Backup Data Penting
Update Sistem Operasi
Menggunakan Antivirus
Tidak Membagikan Kode OTP
